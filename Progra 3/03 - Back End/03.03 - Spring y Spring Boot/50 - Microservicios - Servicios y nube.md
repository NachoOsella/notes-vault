# Microservicios - Servicios y nube

## Definicion
La nube potencia microservicios porque permite aprovisionar y escalar recursos por servicio bajo demanda.

## Explicacion
- *Que problema resuelve*
    Evita sobredimensionar infraestructura para toda la aplicacion.
- *Como funciona por arriba*
    Cada servicio usa recursos administrados (compute, DB, mensajeria, observabilidad) segun necesidad.
- *Que implica / que permite*
    Mejor relacion costo-rendimiento y menos carga operativa al usar servicios gestionados.

## Palabras clave
- cloud
- pago por uso
- escalado
- servicios gestionados
