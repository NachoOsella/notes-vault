# Spring Boot - Que es

## Definicion
Spring Boot es una capa sobre Spring que acelera la creacion de aplicaciones listas para produccion con configuracion minima.

## Explicacion
- *Que problema resuelve*
    Reduce setup manual y decisiones repetitivas de infraestructura inicial.
- *Como funciona por arriba*
    Combina autoconfiguracion, starters y servidor embebido para ejecutar rapido.
- *Que implica / que permite*
    Arrancar proyectos en minutos y enfocarse en logica de negocio.

## Palabras clave
- Spring Boot
- autoconfiguracion
- starter
- servidor embebido

## Comparaciones tipicas
- vs [[03 - Spring - Que es]]: Spring es la base; Boot acelera y opiniona el arranque.
- vs framework sin convencion: Boot aplica "convencion sobre configuracion".

## Errores comunes
- Creer que Spring Boot reemplaza Spring Framework.
- Pensar que no se puede personalizar (si se puede).
