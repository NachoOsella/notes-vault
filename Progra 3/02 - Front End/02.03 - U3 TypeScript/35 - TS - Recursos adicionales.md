# TS - Recursos adicionales

## Definición

Recursos para ampliar o repasar TypeScript con documentación y tutoriales.

## Explicación

- *Qué problema resuelve*
    Tener referencias confiables para profundizar en sintaxis, configuración y ejemplos prácticos.

- *Cómo funciona por arriba*
    - Documentación oficial: referencia completa y actualizada
    - Tutoriales: repaso rápido con ejemplos

- *Qué implica / qué permite*
    - Resolver dudas puntuales (opciones de `tsconfig`, tipos, módulos)
    - Ver ejemplos adicionales fuera de clase

## Recursos

- Documentación oficial de TypeScript: `typescriptlang.org/docs`
- Tutorial TypeScript de W3Schools: `w3schools.com/typescript`

## Palabras clave

- Documentación
- Referencia
- Tutorial

## Comparaciones típicas

- vs [[03 - TS - Introducción a TypeScript]]: la introducción explica el concepto; estos recursos sirven para ampliar.

## Preguntas de examen

- ¿Qué recurso usarías para consultar opciones de `tsconfig.json`?
- ¿Qué ventaja tiene usar documentación oficial frente a un tutorial?

## Errores comunes

- Copiar configuraciones sin entender qué hace cada opción (termina rompiendo el build).
