# Tailwind - Conceptos clave

## Definición
Tailwind CSS es un framework de CSS basado en utilidades que permite construir interfaces de usuario personalizadas de manera rápida y eficiente mediante la composisición de clases predefinidas directamente en el HTML.

## Explicación
- *Qué problema resuelve*
    Elimina la necesidad de escribir CSS personalizado para cada componente, acelerando el desarrollo y facilitando el mantenimiento del código.
- *Cómo funciona por arriba*
    Proporciona una amplia gama de clases utilitarias que representan estilos CSS comunes (como márgenes, colores, tipografía, etc.) que se pueden combinar directamente en el marcado HTML.
- *Qué implica / qué permite*
    Permite a los desarrolladores crear diseños responsivos y personalizados sin salir del HTML, promoviendo un enfoque de diseño basado en componentes y facilitando la colaboración entre diseñadores y desarrolladores.

## Errores comunes
- Confundir Tailwind con un preprocesador CSS.
- Pensar que Tailwind reemplaza completamente la necesidad de CSS personalizado.
- No aprovechar las clases utilitarias y escribir CSS personalizado innecesariamente.
