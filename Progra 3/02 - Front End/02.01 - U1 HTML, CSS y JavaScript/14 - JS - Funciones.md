# JS - Funciones

## Definición
Las funciones en JavaScript son bloques de código reutilizables que realizan una tarea específica. Se definen utilizando la palabra clave `function`, seguidas de un nombre, paréntesis (que pueden incluir parámetros) y llaves que encierran el cuerpo de la función. Las funciones pueden ser invocadas (llamadas) en cualquier parte del código para ejecutar la tarea definida.

## Explicación
- *Qué problema resuelve*
    Permiten organizar el código en bloques lógicos, facilitando la reutilización y el mantenimiento, evitando la repetición de código.
- *Cómo funciona por arriba*
    Al definir una función, se crea un contenedor para un conjunto de instrucciones que pueden ser ejecutadas múltiples veces simplemente llamando a la función por su nombre.
- *Qué implica / qué permite*
    Facilita la modularidad del código, permitiendo dividir tareas complejas en partes más manejables y reutilizables.

## Errores comunes
- Olvidar llamar a la función después de definirla.
- No pasar los argumentos necesarios al llamar a la función.
- Confundir parámetros (definición) con argumentos (llamada).
