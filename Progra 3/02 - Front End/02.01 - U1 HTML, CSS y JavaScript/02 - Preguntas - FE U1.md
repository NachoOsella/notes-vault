## Respuestas (opcional)

### ¿Qué es HTML?
Respuesta: Lenguaje de marcado para estructurar contenido web con etiquetas. Ver: [[03 - HTML - Definición]]

### ¿Cuál es la diferencia entre HTML y CSS?
Respuesta: HTML define la estructura y el contenido; CSS define la presentación visual. Ver: [[03 - HTML - Definición]] y [[06 - CSS - Definición]]

### ¿Cuál es la diferencia entre etiquetas semánticas y no semánticas?
Respuesta: Semánticas aportan significado (mejoran accesibilidad/SEO); no semánticas son genéricas (`div`, `span`). Ver: [[05 - HTML - Etiquetas semánticas]]

### ¿Qué es un selector en CSS?
Respuesta: Un patrón que apunta a elementos HTML para aplicarles estilos (tipo, clase, id, atributo, etc.). Ver: [[07 - CSS - Selectores]]

### ¿Qué es el box model?
Respuesta: Modelo de tamaño/espacio: content + padding + border + margin. Ver: [[09 - CSS - Box Model]]

### ¿Qué es diseño responsivo?
Respuesta: Técnica para que la UI se adapte a distintos tamaños de pantalla usando media queries y layouts flexibles. Ver: [[10 - CSS - Diseño responsivo]]

### ¿Qué es una función en JavaScript?
Respuesta: Bloque reutilizable que recibe parámetros y puede retornar un valor. Ver: [[14 - JS - Funciones]]

### ¿Para qué sirven las estructuras de control?
Respuesta: Controlan el flujo del programa con decisiones (if/switch) y repeticiones (for/while). Ver: [[13 - JS - Estructuras de control]]

### ¿Qué es Tailwind?
Respuesta: Framework CSS utility-first: componés estilos con clases en el HTML en lugar de escribir muchas reglas CSS. Ver: [[11 - Tailwind - Conceptos clave]]

### ¿Cómo se aplica un estilo en Tailwind?
Respuesta: Con clases en el atributo `class`, por ejemplo `class="bg-blue-500 p-4 rounded"`. Ver: [[12 - Tailwind - Sintaxis básica]]
