# HTML - Principales etiquetas

## Definición 
Las etiquetas HTML son elementos fundamentales para estructurar y presentar contenido en la web. Cada etiqueta tiene un proposito específico, entre las más comunes se encuentran `<div>`, `<span>`, `<a>`, `<img>`, `<p>`, `<h1>`-`<h6>`, `<ul>`, `<ol>`, `<li>`, `<table>`, entre otras.

## Explicación 
- *Qué problema resuelve*
    Estructurar y organizar el contenido web de manera semántica y visualmente coherente.
- *Cómo funciona por arriba*
    Las etiquetas HTML se utilizan para definir diferentes tipos de contenido y su función dentro de una página web. Por ejemplo, `<h1>` define un encabezado principal, mientras que `<p>` define un párrafo de texto.
- *Qué implica / qué permite*
    Permite a los desarrolladores web crear páginas estructuradas y accesibles, facilitando la interpretación del contenido por parte de navegadores y motores de búsqueda.

## Comparaciones típicas
- vs [[05 - HTML - Etiquetas semánticas]]: Las etiquetas semánticas proporcionan significado al contenido, mientras que las etiquetas generales como `<div>` y `<span>` son más flexibles pero menos descriptivas.
- vs [[07 - CSS - Selectores]]: HTML define elementos/etiquetas; los selectores son la forma de apuntarlos para aplicar estilos.
