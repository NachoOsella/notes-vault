# CSS - Propiedades comunes

## Definición
Son atributos que definen cómo se verá un elemento HTML en términos de estilo visual. Algunas propiedades comunes incluyen `color`, `font-size`, `margin`, `padding`, `border`, `background-color`, entre otras.

## Explicación
- *Qué problema resuelve*
    Permiten controlar y personalizar la apariencia de los elementos en una página web, mejorando la experiencia del usuario y la estética del sitio.
- *Cómo funciona por arriba*
    Las propiedades se escriben dentro de reglas CSS, asociadas a selectores que apuntan a elementos HTML específicos. Cada propiedad tiene un valor que determina su comportamiento visual.
- *Qué implica / qué permite*
    Permiten una amplia gama de opciones de diseño, desde la tipografía hasta el espaciado y los colores, facilitando la creación de interfaces atractivas y funcionales.

## Errores comunes
- Confundir propiedades similares, como `margin` y `padding`.
- No especificar unidades para valores numéricos (por ejemplo, usar `font-size: 16` en lugar de `font-size: 16px`).
- Olvidar que algunas propiedades requieren valores específicos (por ejemplo, `color` necesita un valor de color válido).
