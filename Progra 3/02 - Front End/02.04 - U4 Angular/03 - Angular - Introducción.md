# Angular - Introducción

## Definición

Angular es un framework frontend de código abierto (mantenido por Google y la comunidad) para construir aplicaciones web modernas, especialmente SPA.

## Explicación

- *Qué problema resuelve*
    Ofrece una estructura completa para aplicaciones grandes, evitando código desordenado y difícil de mantener.

- *Cómo funciona por arriba*
    Se apoya en componentes, servicios, inyección de dependencias, routing, formularios y RxJS para manejar UI y datos de forma organizada.

- *Qué implica / qué permite*
    Permite desarrollar apps escalables y mantenibles con un stack consistente desde el inicio del proyecto.

## Palabras clave

- Framework
- Componentes
- Inyección de dependencias
- SPA
- TypeScript

## Errores comunes

- Confundir Angular (framework) con AngularJS (versión antigua).
- Pensar que Angular solo sirve para apps pequeñas.

## Mini-ejemplo (mental)

Angular es como un “kit completo de construcción”: no solo ladrillos, también planos y reglas para construir más rápido y ordenado.
